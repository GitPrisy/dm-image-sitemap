This plugin was designed to be used on Diariomotor wordpress.

wp dm-image-sitemap generate_all --> generates image sitemaps by months in /sitemaps-images/. directory and image sitemap index in /.

wp dm-image-sitemap generate_month --> regenerate image sitemap from last month with new uploaded images and update index

wp dm-image-sitemap index --> generates index sitemap with all image sitemaps in /.
